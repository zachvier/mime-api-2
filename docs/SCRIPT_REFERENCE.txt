SCRIPT REFERENCE — Mimecast API 2.0 Toolkit
============================================

Run any script from the project root:
  python3 scripts/<script_name>.py

Most scripts take no arguments — just run them. Scripts with interactive
prompts, time-frame pickers, pagination, or CLI args are called out below.

All record-producing scripts share the same vertical output format
(table_utils.print_records) — labeled key: value blocks, one per record.


ACCOUNT & IDENTITY
------------------
  get_account.py                  Account info (add account_code to credentials.txt from output)
  get_support_info.py             Support contacts, renewal date, package
  get_emergency_contact.py        Emergency contact (v1 API)
  get_whoami.py                   Identity info for the authenticated caller (v1 API)
  get_dashboard_notifications.py  Admin dashboard notifications (filtered to last 2 years).
                                  Requires account_code in credentials.txt.


GATEWAY — LOGS WITH TIME FRAMES
-------------------------------
Interactive prompts on these: time frame (1h / 24h / 7d / 30d / custom)
and results-per-page. Paginates interactively.

  get_rejection_logs.py           Gateway rejection logs (admin scope)
  get_held_release_logs.py        Held/release action logs
  get_hold_message_list.py        Currently-held messages (also prompts admin y/n)
  get_audit_events.py             Audit events (adds 60-day option)


GATEWAY — OTHER
---------------
  get_hold_summary_list.py        Counts of currently held messages grouped by policy (aggregate)
  get_email_queues.py             Inbound/outbound queue depth over the last 24h (aggregate)
  get_gateway_details.py          Cloud Gateway outbound config, mail platforms, SPF/MX details
  get_outbound_ip_addresses.py    Configured outbound IP addresses for the account mail platform
  get_email_statistics.py         Cloud Gateway email statistics / delivery health


DLP & TTP LOGS
--------------
  get_dlp_logs.py                 DLP event logs (interactive time frame + pagination)
  get_ttp_url_logs.py             TTP URL click logs
                                  Extra prompts: scanResult (all/clean/malicious), route (all/inbound/outbound)
  get_ttp_attachment_logs.py      TTP attachment protection logs (last 7 days, no prompts)
  get_ttp_impersonation_logs.py   TTP impersonation protection logs (last 7 days, no prompts)


DIRECTORY & DOMAINS
-------------------
  find_groups.py                  Directory groups. Prompts: source (cloud/ldap), optional query, pagination
  get_group_members.py            Group members for a directory group. Optionally lists groups first,
                                  then prompts for the group ID.
  get_internal_domains.py         Internal domains on the account
  get_internal_users.py           Internal users for a domain. Lists internal domains to pick from
                                  (or enter one manually), then paginates results per page
  get_directory_connections.py    Directory connectors configured on the tenant
  get_user_aliases.py             Aliases associated with a primary email address.
                                  Prompts for the email address.
  get_user_attributes.py          Custom attributes registered for a user. Prompts for
                                  the email address. If a thumbnailPhoto attribute is
                                  returned, offers to save it as <email>_thumbnail.jpg
                                  in the current working directory.
  find_delegate_users.py          Delegate users for a primary email address.
                                  Prompts for the primary address.
  get_most_used_contacts.py       Most-used contacts synced from Azure Active Directory.


URLs & ARCHIVE
--------------
  get_all_managed_urls.py         Managed URL list (allow/block entries)
  get_archive_search_logs.py      Archive search activity logs
  get_audit_categories.py         Available audit event category IDs/names
  get_provisioning_packages.py    Provisioning packages available on the tenant


URL DECODER (argument or interactive)
--------------------------------------
  decode_url.py

  With argument:
    python3 scripts/decode_url.py "https://url.us.m.mimecastprotect.com/..."

  Without argument:
    python3 scripts/decode_url.py
    # Prompts for the URL


SHARED HELPERS
--------------
  auth.py           OAuth client_credentials token fetch; reads credentials.txt,
                    project-root .env, or environment variables
  table_utils.py    print_records (vertical key: value output), print_table, offer_more_views


CREDENTIALS FILE (scripts/credentials.txt)
------------------------------------------
  client_id=...
  client_secret=...
  account_code=...   # optional; required by get_dashboard_notifications and
                     # sent as x-mc-accountcode header by get_whoami when present
